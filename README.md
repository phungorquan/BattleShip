# BattleShip
We use two Nuvoton boards (Numaker) to remake BattleShip game.

# Devices:
+ Board : NUC131(Numaker)
+ LCD: ST7920 (128x64)
+ Ethernet: ENC28J60

# Operating
+ Run on FreeRTOS (but it can not receive data now)

# Soft
+ Keil C V5.17
+ Proteus 8.6
+ WireShark (Check packet)
+ Hercules (Receive and Send packet)

# Protocol
+ UDP
